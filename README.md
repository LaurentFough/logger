# logger

Empower your log capabilities to the maximum level

## Install

With [fisherman]

```
fisher logger
```

## Usage

```fish
log foo
debug foo # or log foo 1
warn foo # or log foo 2
error foo # or log foo 3
fatal foo # or log foo 4
critical foo # or log foo 5
```
